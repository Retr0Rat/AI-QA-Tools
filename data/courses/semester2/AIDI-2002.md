## Overview
Computer vision enables machines to interpret and understand visual information. This course covers image processing fundamentals, classical CV techniques, and modern deep learning approaches. Students build vision pipelines for detection, segmentation, and generation tasks using industry tools.

## Tools & Technologies
- Python 3.x
- OpenCV
- TensorFlow 2.x / Keras
- PyTorch
- torchvision
- Ultralytics YOLO (v8)
- Roboflow (dataset management and annotation)
- Albumentations (image augmentation)
- Weights & Biases
- Google Colab (GPU)
- Pillow (PIL)
- Matplotlib

## Topics
- Image representation, color spaces, and filtering
- Edge detection, morphological operations, and feature descriptors (SIFT, ORB)
- Image classification with CNNs
- Data augmentation strategies
- Object detection: YOLO, Faster R-CNN, SSD
- Semantic segmentation (U-Net, DeepLab)
- Instance segmentation (Mask R-CNN)
- Pose estimation
- Video analysis and optical flow
- Image generation with GANs and diffusion models (introductory)
- Dataset annotation and management

## Projects & Capstone

### Project 1: Custom Object Detector
Students collect or source a dataset (minimum 500 images), annotate it using Roboflow, and fine-tune a YOLOv8 model. The trained detector is evaluated and demonstrated on a video stream or test set.

### Project 2: Image Segmentation Application
Build a semantic segmentation pipeline for a domain-specific task (e.g., road segmentation for autonomous driving, medical image tissue segmentation) using U-Net with a pre-trained encoder.
