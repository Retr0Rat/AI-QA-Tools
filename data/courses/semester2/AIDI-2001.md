## Overview
This course covers the full spectrum of NLP, from linguistic preprocessing through classical ML approaches to modern transformer-based language models. Students implement NLP systems for real tasks and develop an understanding of how large language models are trained and used.

## Tools & Technologies
- Python 3.x
- NLTK (Natural Language Toolkit)
- spaCy
- Hugging Face Transformers (BERT, RoBERTa, DistilBERT)
- Hugging Face Datasets
- Gensim (Word2Vec, FastText)
- scikit-learn (text features)
- PyTorch
- Weights & Biases
- Jupyter Notebooks

## Topics
- Text preprocessing: tokenization, lemmatization, stemming, stopword removal
- Bag-of-Words, TF-IDF, n-gram representations
- Word embeddings: Word2Vec, GloVe, FastText
- Text classification and sentiment analysis
- Named Entity Recognition (NER)
- Part-of-speech tagging and dependency parsing
- Sequence-to-sequence models
- Attention and the Transformer architecture (deep dive)
- BERT and fine-tuning pre-trained language models
- Machine translation fundamentals
- Question answering and information extraction
- Ethical issues: bias in language models, misinformation

## Projects & Capstone

### Project 1: Sentiment Analysis Pipeline
Build a sentiment analysis system for product reviews. Students compare classical ML (TF-IDF + logistic regression) vs. fine-tuned BERT, analyse performance differences, and document findings.

### Project 2: NER and Information Extraction
Train a named entity recognition model on a domain-specific corpus and use it to extract structured information (e.g., extracting drug names and dosages from clinical text or extracting company/role pairs from job postings).
