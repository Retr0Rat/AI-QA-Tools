## Overview
Generative AI is reshaping industries. This course explores the architecture and training of LLMs, prompt engineering techniques, retrieval-augmented generation (RAG), fine-tuning strategies, and responsible deployment. Students build end-to-end LLM-powered applications.

## Tools & Technologies
- Python 3.x
- OpenAI API / Anthropic Claude API
- Hugging Face Transformers and PEFT
- LangChain
- LlamaIndex
- ChromaDB / Pinecone (vector databases)
- Sentence Transformers (embeddings)
- Weights & Biases
- Streamlit / Gradio (rapid UI prototyping)
- FastAPI
- Docker

## Topics
- Transformer architecture deep dive (GPT, BERT, T5)
- Pre-training, instruction tuning, and RLHF
- Prompt engineering: zero-shot, few-shot, chain-of-thought
- Retrieval-Augmented Generation (RAG) architecture
- Vector databases and semantic search
- Fine-tuning and parameter-efficient fine-tuning (LoRA, QLoRA)
- LLM evaluation and benchmarking
- Agents and tool use with LLMs
- Hallucination, safety, and alignment
- Multimodal models (text + image)
- Cost, latency, and scaling considerations

## Projects & Capstone

### Project 1: RAG Application
Students build a domain-specific Q&A application using RAG: ingest a document corpus, generate embeddings, store in ChromaDB, and use an LLM to answer natural language questions with cited sources.

### Project 2: Fine-Tuned LLM
Fine-tune a small open-source LLM (e.g., Mistral 7B, LLaMA 3) on a custom dataset using QLoRA. Students evaluate the base vs. fine-tuned model on a held-out test set and document the training process.
