## Overview
This course covers the algorithmic foundations of AI, from classical search and planning to supervised and unsupervised machine learning algorithms.

## Topics
- Search algorithms
- Optimization techniques
- Supervised learning
- Unsupervised learning
- Model evaluation

## Tools
- Python
- scikit-learn
- NumPy
- Pandas

## Projects
- Classification and clustering assignments
- Algorithm comparison studies
