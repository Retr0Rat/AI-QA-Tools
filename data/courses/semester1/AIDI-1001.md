## Overview
Students learn to transform raw data into meaningful visual stories. The course covers data visualization principles, dashboard design, and tools used by data professionals.

## Topics
- Data visualization principles
- Dashboard design
- Statistical storytelling
- Visual communication of AI insights

## Tools
- Tableau
- Python (matplotlib, seaborn)
- Excel

## Projects
- Canadian Immigration Data Analysis with Tableau dashboards
